Wikipedia Treasure Hunt

Having completed Codecademy courses on how to create interactive websites with JavaScript and jQuery and having done some self-study I decided to use my new-found knowledge to create an online quiz where each answer would only be revealed by correctly answering the previous question. This was created for a friend, but I include it here to show how I took the knowledge I had gained and applied it to a new problem.
